Контрольні питання#

  1. До форматів мультимедійних даних належить
    • gif. jpg. png
    • html. mp3
    • avi. mov. htm
    • avi. mov. mpeg. wav
  2. Атрибут controls відповідає за:
    • тип мультимедійного вмісту
    • елементи керування мультимедійним програвачем
    • джерело файлу мультимедійного вмісту
    • ширина мультимедійного програвача
  3. Висоту та ширину графічного об’єкта, розміщеного на веб-сторінці, можна задати:
    • за допомогою атрибутів WIDTH та HEIGHT у тегу BODY.
    • за допомогою атрибутів WIDTH та HEIGHT у тегу IMG;
    • за допомогою атрибута SRC;
    • за допомогою атрибутів WIDTH та HEIGHT у тегу ALT;
    • у дужках після URL-адреси файлу зображення;
  4. Який атрибут тегу <audio> вказує шлях до аудіофайлу?
    • autoplay
    • controls
    • src
    • preload
  5. Чи можна відтворювати аудіо- та відеофайли прямо у вікні браузера?
    • Так
    • Ні
    • Тільки в окремих випадках
    • Тільки про спеціальних налаштуваннях браузера
  6. Який тег HTML використовується для вказівки кількох медіафайлів?
    • <source>
    • <picture>
    • <audio>
    • <video>
  7. Авторське право — це право на…
    • зазначення імені автора (або заборона зазначення) на створеному об’єкті та його примірниках
    • вибір псевдоніма та зазначення його замість справжнього імені автора на створеному об’єкті та його примірниках
    • збереження цілісності створеного об’єкта та на заборону будь-якої його зміни
    • дозвіл (заборону) використовувати (друкувати, копіювати, перекладати, виконувати, демонструвати тощо) об’єкт іншим особам тощо
  8. Об’єктами мультимедіа являються…
    • текстові написи
    • відеофайли
    • таблиці
    • бази даних
    • аудіофайли
    • графічні об’єкти
  9. Комп’ютеризована технологія, що об’єднує роботу зі всіма джерелами даних, засіб подання різних видів інформації у цифровому вигляді - це …
    • кроссбраузерність
    • мультимедіа
    • адаптивність
    • анімація
  10. Добавити відео на веб-сторінку можна використавши тег…
    • <img>
    • <video>
    • <title>
    • <sourse>
  11. Щоб вставити малюнок на веб-сторінку, необхідно скористатися тегом:
    • <img src = "...">
    • <img = "...">
    • <src img = "...">
    • <img css = "...">
  12. Вкажіть графічні елементи веб-дизайну
    • Фотографії
    • Рисунки
    • Фонові малюнки
    • Заголовки
    • Кнопки
    • Списки
  13. Основні формати графічних файлів у веб-дизайні
    • png
    • jpg
    • bmp
    • svg
    • gif
    • tiff
  14. Обов’язковим атрибутом тегу для вставки зображення є
    • img
    • src
    • image
    • link
    • width
    • height
  15. Якщо не задавати розміри зображення:
    • Зображення відобразиться на сторінці в реальному розмірі
    • Браузер потребуватиме часу на те, щоб дізнатися розміри і завантажити зображення.
    • Браузер спочатку зарезервує місце під зображення, підготує макет документа, відобразить текст і лише потім завантажить зображення.
    • Розмір буде обчислюватися автоматично для збереження пропорцій малюнка
  16. Об’єктом авторсько-правових відносин є …
    • нематеріальне благо у вигляді продукту духовної творчості
    • твори науки, літератури, мистецтва
    • матеріальне благо у вигляді продукту
  17. Цифровий водяний знак -
    • Дозвіл, та (заборону) використовувати (друкувати, копіювати, перекладати, виконувати, демонструвати тощо) об’єкт іншим осібам тощо
    • Технологія, створена для захисту авторських прав мультимедійних файлів.
    • Результат творчої діяльності однієї людини (автора, виконавця, винахідника та інш.) або кількох осіб.
  18. Яке посилання є правильним для вставлення зображення на веб сторінку?
    • <img id="image">
    • <img src="image.png">
    • <img type="image">
    • <img src="image">
  19. Виконання якої дії забезпечить даний тег <img src="image.jpg">?
    • додавання таблиць
    • додавання картинок, фотографій, ілюстрацій, схем
    • додавання списків
  20. Для додавання гіперпосилань на веб-сторінку використовують тег:
    • <image>
    • <href>
    • <img>
    • <a href>...</a>
  21. До мультимедійних об’єктів належать…
    • аудіооб’єкти
    • відеооб’єкти
    • графічні зображення (рухомі й нерухомі)
    • Жести
  22. Яким тегом файл малюнка foto.jpg вставити на веб-сторінку?
    • <img src="my.jpg">
    • <html src="foto.jpg">
    • <img "foto.jpg"=arc>
    • <img src="foto.jpg">

    • всі права захищено
    • копіювайовані дані
    • викрадення чужого матеріалу
    • написання унікального тексту на основі поданого
  24. Додати відео на веб-сторінку можна використавши тег…
    • <img>

    • <sourse>
